# 神经突触 — 认知挑战

测试人类四大认知维度的网页小游戏，共 20 关，由易到难，逐关解锁。

**在线体验 →** https://asterisk622.github.io/neuro-synapse-game/

## 测试类别

| 类别 | 说明 | 关卡数 |
|------|------|--------|
| ⚡ 反射弧测试 | 刺激-反应协调能力：快速点击、颜色筛选、动态判断、双重刺激、色键反应 | 5 |
| 🧠 短时记忆测试 | 工作记忆容量：数字记忆、网格记忆、逆序回忆 | 5 |
| 🔮 逻辑思维测试 | 推理与模式识别：数列补全、图形匹配、三段论、逻辑推理 | 5 |
| 🎯 反应速度测试 | 纯粹速度：基础反应、方向反应、斯特鲁普测试、混合刺激 | 5 |

## 特性

- 单文件 HTML，无需构建，浏览器直接打开
- 暗色科幻/神经元主题，Canvas 粒子背景动画
- Web Audio API 音效反馈
- localStorage 自动保存进度
- 1-3 星评级系统
- 响应式布局，支持桌面和移动端

## 本地运行

```bash
# 任选一种方式
open index.html
# 或
python3 -m http.server 8080
```
